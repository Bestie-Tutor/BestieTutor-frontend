#### How to Install

<pre>
git clone https://github.com/Bestie-Tutor/web-frontend.git
cd web-frontend
yarn
yarn dev
</pre>

#### Structure

* src/components: 작은 단위의 컴포넌트가 담긴 폴더
* src/pages: 실제 각 페이지를 위한 컴포넌트
